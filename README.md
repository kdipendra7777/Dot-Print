# DotPrint

DotPrint is a modern and creative printing platform focused on delivering high-quality digital printing solutions with a clean, premium, and minimal experience.

## Features

- Modern Minimal UI
- Premium Branding Experience
- Responsive Design
- Creative Product Showcase
- Smooth Animations
- Mobile Friendly Layout
- Professional Typography

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js

## Vision

DotPrint aims to create a next-generation digital printing experience that feels modern, creative, and professional while keeping the interface simple and accessible for everyone.

Currently under active development
